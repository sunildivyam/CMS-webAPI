loader
directive in module raiweb.core

Description

Displays a progress loader modal to parent or body

Usage

as element:
<loader
       at-body="{boolean}"
       is-loading="{boolean}"
       [message="{string}"]>
</loader>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
atBodyboolean

defaults to "false". A "false" value specifies that loader should be modal to immidiate relative parent. A "true" value specifies that loader should be modal to body.

isLoadingboolean

toggles the loader

message
(optional)
string

loading message string
